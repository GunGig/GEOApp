# GEOApp - 品牌智能监测与自动化采集系统

GEOApp 是一款专为品牌舆情公关、电商运营及市场分析人员打造的 **GEO（Generative Engine Optimization，生成式引擎优化）监测与数据采集客户端**。

系统可模拟人工在各大主流大模型 AI 平台进行真实提问，自动化采集深度思考回答、参考来源网址及分享链接，并将结构化数据与卡片截图一键写回至飞书多维表格，极大提升了品牌心智占有率监测的效率。

---

## 🌟 核心特性

- **🤖 多大模型支持**：原生驱动 **豆包 (Doubao)**、**腾讯元宝 (Yuanbao)**、**DeepSeek**、**文心一言 (Wenxin)** 等主流对话界面。
- **📊 飞书多维表格无缝对接**：自动解析飞书多维表格任务队列，并将采集到的 AI 回答、引用网址、对话标题、拓展问法等自动批量写回至指定数据表。
- **📸 智能卡片高亮截图**：在自动生成的会话中，自动高亮品牌自有产品，并伸展页面截取超长、高清的对话卡片全图，统一采用标准日间模式风格。
- **🔑 一次登录，自动验证**：支持用户数据本地沙盒化隔离，预登录后自动保存 Cookie 状态，下次启动免验证直接后台无感采集。
- **🛡️ 稳定避坑机制**：针对各大模型频繁更新的选择器、防机器人机制、剪贴板读写限制进行了底层优化，保障长期稳定运行。

---

## 🖥️ 运行环境要求

- **操作系统**：macOS 11.0 (Big Sur) 及以上（支持 Apple Silicon M1/M2/M3 及 Intel 芯片）
- **依赖软件**：
  - 系统需安装 **Google Chrome** 浏览器（建议更新至最新版）。
  - 无需额外下载 ChromeDriver，系统将自动管理驱动。

## 📦 下载与安装指南

### 1. 下载安装包
请前往本仓库右侧的 [Releases 页面](https://github.com/您的用户名/您的仓库名/releases) 下载最新的 `.dmg` 格式安装程序：
- 📥 **[点击下载最新版 GEOApp (macOS)](https://github.com/您的用户名/您的仓库名/releases/latest)**

### 2. 安装步骤
1. 双击下载的 `GEOApp.dmg` 文件。
2. 在弹出的窗口中，将 **GEOApp** 图标拖拽到右侧的 **Applications (应用程序)** 文件夹。
3. 在应用程序中找到 GEOApp 双击打开。

> [!IMPORTANT]
> **关于安全拦截提示（已损坏/无法打开）的解决办法**：
>
> 由于 macOS 严格的安全隐私限制，首次打开未在 App Store 上架的第三方应用时，可能会提示“无法打开”或“已损坏”。
> 请打开终端（Terminal），输入以下命令并回车，即可正常运行：
> ```bash
> sudo xattr -r -d com.apple.quarantine /Applications/GEOApp.app
> ```

## ⚙️ 快速使用流程

1. **配置飞书连接**：在主界面输入您的飞书应用授权码、飞书多维表格的分享链接。
2. **预登录 AI 账号**：在“预登录管理”中，一键打开对应 AI 平台的浏览器，扫码/手机号登录一次，等待检测成功并自动关闭。
3. **开始自动监测**：点击主界面的“开始监测”，GEOApp 将在后台自动启动 Chrome 执行全自动问答、高亮、截图、正文抓取以及飞书表格数据自动写回。
4. **查看运行记录**：在界面日志和飞书表格的“运行记录表”中可实时追踪成功与失败明细。
